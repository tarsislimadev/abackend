## how to install

```sh
# bash

npm i @brtmvdl/backend
```

## how to use

### package.json

```json
{
  "type": "module",
  "scripts": {
    "start": "node index.js"
  },
  ...
}
```

### index.js

```js
import { Server } from '@brtmvdl/backend'

const server = new Server()

server.get('/', (req, res) => res.setJSON({ id: Date.now() }))

server.listen(80)
```

### node start

```sh
# bash

npm start
```
